Lab 27

One of the dialog's greatest assets is modality. This feature creates an overlay that covers the underlying page below the dialog while it is open. The overlay is removed as soon as the dialog is closed. None of the underlying page content can be interacted with in any way while the dialog is open.

Dialog

A dialog has a lot of default behaviour built-in, but few methods are needed to control it programmatically, making this an easy-to-use widget that is also highly configuable and powerful.

Accordion

We can easily place other UI widget into the dialog, such as the accordion widget
